LIONHEART

Lionheart was a brand born of a parent Not-for-profit organisation that assists adolesence with learning difficulties. Lionheart itself is an event for learning about inclusivity from industry leaders. It is for students as well as teachers.

This project was to design a brandmark for Lionheart, as well as other assets that would be needed for the organisation.

BRANDMARK

The main requirement for this project was a strong brandmark. The brief stated that it needed to be a combination mark of a pictorial and word mark, as it is more inclusive and easier to identify. The pictorial mark was designed to encompass the brand name in a bold, strong & nobel compesition to reflect the core values of the organisation.  

PROCESS

First starting with the sketches, I quickly sketched out some idea's on how to encorperate the lion & the heart together. After giving myself some inspiration and a foundation to start from. I ideated several different marks to send off to the founder.
